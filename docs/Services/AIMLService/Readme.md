# AI/ML Service 
## Giới thiệu

Service AI/ML cung cấp các chức năng trí tuệ nhân tạo và học máy cho hệ thống CityResQ360.

## Chức năng chính

### 1. Phân tích dữ liệu

- Xử lý và phân tích dữ liệu từ các nguồn khác nhau
- Dự đoán và phát hiện các mẫu bất thường

### 2. Machine Learning Models

- Huấn luyện và triển khai các mô hình ML
- Tối ưu hóa hiệu suất mô hình

### 3. API Endpoints

```typescript
// Ví dụ sử dụng AI/ML Service
POST / api / aiml / predict;
POST / api / aiml / analyze;
GET / api / aiml / models;
```

## Tích hợp

Service này tích hợp với:

- **AnalyticsService**: Cung cấp dữ liệu phân tích
- **CoreAPI**: Xử lý các yêu cầu API
- **IncidentService**: Dự đoán và phân loại sự cố
